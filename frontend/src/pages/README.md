# Pages

Page-level components corresponding to routes.

## Structure

```
pages/
├── auth/           # Authentication pages (Login, Register)
├── dashboard/      # Dashboard home pages (per role)
├── homework/       # Homework management pages
├── submissions/    # Submission viewing and grading
├── textbooks/      # Textbook management
├── students/       # Student management
├── analytics/      # Analytics and reports
└── settings/       # User settings
```

## Naming Convention

- PascalCase for page files
- Page suffix (e.g., LoginPage.tsx)
