# Components

Reusable UI components organized by category.

## Structure

```
components/
├── ui/           # Base UI elements (Button, Input, Card, etc.)
├── forms/        # Form components (LoginForm, HomeworkForm, etc.)
├── layout/       # Layout components (Header, Sidebar, Footer)
├── data/         # Data display (Table, List, Stats)
└── feedback/     # Feedback components (Modal, Toast, Spinner)
```

## Naming Convention

- PascalCase for component files
- index.ts for barrel exports
- Component.test.tsx for tests
