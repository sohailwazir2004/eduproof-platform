# Skills Reference for Claude Code - EduProof Project

## Full-Stack Web Development
- React + Vite + Tailwind CSS
- Component-based design & dashboards
- State management (Context API / Redux / Zustand)
- Responsive & mobile-friendly UI
- Form handling and validation

## Mobile Development
- React Native + Expo
- Mobile navigation (React Navigation)
- Camera integration and file uploads
- Push notifications (Firebase Cloud Messaging)
- Cross-platform deployment (Android + iOS)

## Backend Development
- Python + FastAPI
- RESTful API development
- JWT-based authentication
- Role-Based Access Control (Student, Teacher, Parent, Principal)
- Async API handling
- File handling for images and PDFs
- Modular backend architecture

## Database & Storage
- PostgreSQL
- SQLAlchemy ORM
- Data modeling (Users, Roles, Classes, Subjects, Homework, Submissions)
- Cloud storage: AWS S3 / Cloudinary
- Private and secure file URL management

## AI / Machine Learning
- OCR for handwritten homework (Tesseract / Google Vision API)
- AI content analysis for homework relevance
- Duplicate / plagiarism detection
- AI-generated summaries for teacher dashboard
- Image preprocessing and compression

## Cloud & Deployment
- Environment variable configuration (.env)
- Deployment platforms:
  - Backend: Render / Railway / AWS EC2
  - Frontend: Vercel / Netlify
  - Mobile: Expo App Store / Play Store
- Notifications integration via Firebase
- CI/CD pipeline basics

## Testing & Quality Assurance
- Unit tests for backend models
- API integration tests
- Frontend & mobile component tests
- Test-driven development best practices

## Tools & Workflow
- Git / GitHub for version control
- Agile / Kanban workflow
- Modular folder structure design
- Documentation best practices (README.md, claude.md)
- Task breakdown and incremental development
